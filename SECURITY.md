# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in OpenPersona, please report it responsibly.

**Email:** security@example.com

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

You should receive a response within 48 hours. We appreciate your help in keeping this project secure.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.7.x   | Yes       |
| < 0.7   | No        |
